## 一、数据源方案

### 1.1 行情数据（最核心，立刻可用的免费方案）

**推荐：AkShare**（GitHub 21k star，Python 一行安装）

```python
pip install akshare
```

| 数据类型 | AkShare 函数示例 | 说明 |
|---------|-----------------|------|
| A 股日线行情 | `ak.stock_zh_a_hist(symbol="000001", period="daily")` | 免费，含开高低收成交量 |
| A 股实时行情 | `ak.stock_zh_a_spot_em()` | 全市场快照 |
| 指数数据 | `ak.stock_zh_index_daily(symbol="sh000001")` | 上证/深证/创业板 |
| 北向资金 | `ak.stock_hsgt_north_net_flow_in_em()` | 北上资金流向 |
| 龙虎榜 | `ak.stock_sse_lhb()` | 每日龙虎榜数据 |
| 美股行情 | `ak.stock_us_spot_em()` | 美股实时行情 |

**备选：Tushare**
- 免费版 token 能拿到大部分行情，Pro 版需要积分
- 优点是数据更规范、文档更好
- 注册地址：https://tushare.pro

### 1.2 新闻/公告数据

| 方案 | 说明 | 接入难度 |
|------|------|---------|
| **东方财富新闻接口** | 可通过 AkShare 的 `ak.stock_info_news_em()` 获取 | 低 — 一行代码 |
| **新浪财经新闻** | `ak.stock_info_news_sina()` | 低 |
| **公司公告** | `ak.stock_notice_report()` | 低 |

### 1.3 财报数据

| 方案 | 说明 |
|------|------|
| **AkShare 财报接口** | `ak.stock_financial_abstract(stock="000001")` 获取利润表/资产负债表 |
| **Tushare 财报** | `pro.fina_mainbz()` 更完整的财务数据 |

### 1.4 社交舆情（可选扩展）

| 方案 | 说明 |
|------|------|
| **雪球** | 雪球帖子/评论爬虫，需要处理反爬 |
| **微博/股吧** | 东方财富股吧评论，可用爬虫获取 |
| **微信公众平台** | 难度较高，暂时不建议 |

## 二、回测框架方案

### 2.1 backtrader（推荐，入门首选）

```python
pip install backtrader
```

- 最流行的 Python 回测框架
- 上手简单，文档完善
- 支持多策略、多数据源、滑点/佣金模拟
- 自带绘图功能

**最简示例（5分钟就能跑起来）：**
```python
import backtrader as bt

class SmaCross(bt.SignalStrategy):
    def __init__(self):
        sma1, sma2 = bt.ind.SMA(period=10), bt.ind.SMA(period=30)
        self.signal_add(bt.SIGNAL_LONG, bt.ind.CrossOver(sma1, sma2))

cerebro = bt.Cerebro()
data = bt.feeds.PandasData(dataname=df)  # df 是 AkShare 拉的数据
cerebro.adddata(data)
cerebro.addstrategy(SmaCross)
cerebro.run()
cerebro.plot()
```

### 2.2 vnpy（进阶选项）

- 全功能量化交易平台，内置回测+实盘
- 支持多家券商接口
- 学习曲线比 backtrader 陡，适合有经验后迁移

## 三、LLM 集成方案

### 3.1 新闻情绪分析（最简单的 LLM 切入）

```
新闻文本 → LLM 分析情绪（正面/负面/中性 + 置信度） → 生成情绪因子分数
                                                         ↓
                                                 接入回测/策略
```

**用你的 LLM 或者 API：**
```python
def analyze_sentiment(news_text):
    prompt = f"分析以下金融新闻的情绪倾向，输出 正面/负面/中性 和置信度(0-1)：\n{news_text}"
    response = llm_api(prompt)  # 你的 LLM
    return parse_sentiment(response)
```

## 四、交易执行方案

### 4.1 Phase 1：模拟盘/回测（暂不需要券商接口）

- 用 backtrader 跑回测
- 模拟交易：有一个虚拟仓位管理模块，接收信号但不实际下单

### 4.2 Phase 2：实盘（需要券商接口）

| 券商 | 接口方案 | 说明 |
|------|---------|------|
| **华泰** | xtquant（官方 Python SDK） | 支持 A 股，文档完善 |
| **富途** | Futu OpenAPI（Python SDK） | 支持港股/美股/A股 |
| **东方财富** | 同花顺/迅投接口 | 需券商支持 |
| **IB（盈透）** | ib_insync（Python） | 国际化，支持多市场 |

建议：Phase 1 先不需要管券商接口，回测跑通了再研究。

## 五、阶段性执行计划

### Phase 1：数据管道 + 基础回测（1-2周）

| 步骤 | 具体 | 交付 |
|------|------|------|
| 1. 搭环境 | pip install akshare backtrader pandas matplotlib | 环境就绪 |
| 2. 拉数据 | 用 AkShare 拉一只股票（比如茅台/平安）的历史数据 | 有 DataFrame 数据 |
| 3. 跑回测 | 写一个简单的均线策略，在 backtrader 上跑通 | 看到回测曲线 |
| 4. 加简单因子 | 加几个基本技术指标（RSI, MACD）看效果 | 多策略对比 |

### Phase 2：加 LLM 分析层 + 人机协作面板（3-4周）

| 步骤 | 具体 | 交付 |
|------|------|------|
| 1. LLM 情绪分析 | 拉新闻，用 LLM 做情绪打分，生成因子 | 情绪因子加入回测 |
| 2. 综合策略 | 传统因子 + LLM 因子融合 | 带 AI 分析的策略 |
| 3. 人类决策面板 | 做通知/确认机制（飞书 bot 或 Web 面板） | 人可以确认/修改后执行 |
| 4. 复盘系统 | 自动生成每日复盘报告 + 对话式追问 | 每天自动出报告 |

### Phase 3：优化 + 自主交易（5周+）

| 步骤 | 具体 |
|------|------|
| 1. 策略参数优化 | 网格搜索/贝叶斯优化 |
| 2. 回测验证稳定 | 多周期、多股票验证 |
| 3. 接入实盘 | 选券商，开通接口，小资金试水 |
| 4. 跟踪迭代 | 实盘数据反馈到LLM，持续学习 |
