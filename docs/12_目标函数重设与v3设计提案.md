# 12 · 目标函数重设与 v3 设计提案（收益率 × 降风险）

> yquant 项目文档包 · 编号 12 · v1.0 提案稿 · 2026-07-05
> 委托人新指令：不考虑 token 成本、不考虑税务，目标只有两个——**收益率**与**降低风险**；允许推翻既往一切决策（含 v2.2 的美/港切换）。
> 本篇性质：**战略提案**，经委托人拍板后由 §7 的修订指令并入 03/06/08/09/10。写作立场延续 02 篇：每个结论给出正反证据，被推翻的旧决策注明推翻理由，证据不足处如实标注。

## 1. 目标函数的严格化（先把"只要收益和降 risk"翻译成工程语言）

**目标**：最大化长期净值口径的风险调整收益（主指标：卡玛比率 = 年化收益/最大回撤；辅指标：夏普、月度亏损尾部）。**净值口径意味着交易成本与滑点照常建模**——它们不是税，是真实损耗；被移除的仅是：LLM 调用预算上限（原 ¥2/日删除，记账保留）与税务建模（预扣税/资本利得均置零）。

**放开约束的诚实备案**：税务在跨境投资中真实影响净收益（记录在案，风险自担，成本模型保留 `tax_*` 参数钩子且默认 0）；token 无限化改变的不是"LLM 能不能预测"，而是"能把学术界验证过的 LLM 用法开到多大规模"。

**"降低风险"的可操作定义**：①最大回撤显著低于满仓单一股市；②不承受可分散的集中度风险（单市场/单风格/单币别的暴露有上限）；③极端行情有预先定义的自动减险动作，而非临场决定。

## 2. 证据综述（广泛查询的结果，三条证据线）

### 2.1 降险工具线：什么是靠得住的，什么是被夸大的

**多资产分散**是唯一没有争议的免费午餐：股/债/黄金/大宗的长期相关性结构使组合回撤远小于单一股市，这是全部机构配置实践的地基，无需重复论证。

**趋势跟踪/时序动量**：跨百年、跨数十个市场的证据显示其最稳健的贡献是**截断左尾**（危机期减仓避险），Faber 式 10 月均线择时历史上把股市 50% 级别回撤压到 20% 级别；代价同样明确——震荡市的反复止损（whipsaw）与 V 型反弹中的踏空（2020 年 3–4 月是教科书案例），因此对**收益率**的贡献不稳定，对**回撤**的削减稳健。

**波动率管理**：Moreira–Muir（2017）报告正 alpha，但 Cederburg 等（2020，JFE）用 103 个策略检验后发现波动率管理组合并不系统性跑赢原始组合，其隐含策略在实时中难以实施，样本外合理版本的夏普普遍更低；同一文献流亦确认其**危机期降暴露、削回撤**的机制真实存在。→ 设计裁决：我们**只向波动率目标索取"回撤控制"，不向它索取"收益增强"**，且只做减仓方向（无杠杆），从而避开其争议最大的部分。

### 2.2 LLM-alpha 线：token 无限化真正解锁的东西

**新闻研判**：Lopez-Lira & Tang 用知识截止**之后**的头条新闻（2021.10–2024.5）做严格样本外检验：GPT-4 对新闻的即时市场反应判断达到约 90% 的组合日命中率（该初始反应本身不可交易），其评分对**后续漂移**有显著预测力，在小盘股与负面新闻中最强；同时作者明确记录**策略收益随 LLM 普及而下降**——这是一条正在被套利掉的边。

**财报研判**：Kim–Muhn–Nikolaev（芝大 Booth）把匿名化、标准化的财务报表交给 GPT-4 做链式推理，方向性预测下一年盈利变化的准确率约 60.4%，超过人类分析师约 7 个百分点，与专门训练的神经网络持平；匿名化设计排除了"背答案"，基于其预测构建的组合获得更高的夏普与 alpha。

**反面证据**（必须同权重呈现）：Glasserman–Lin 证明用 GPT 情绪做**截止日前**回测存在前视偏差（模型"记得"历史）；Sarkar–Vafa 与 Lopez-Lira 等 2025 年工作进一步确立"必须只用截止后样本"的方法论铁律；Drinkall 等发现信用评级预测上传统方法仍胜过生成式 LLM——LLM 不是万能预测器，它的可证实优势集中在**文本→基本面方向**这类信息处理任务上。

**综合裁决**：LLM alpha 真实存在但有四个附带条款——幅度有限、正在衰减、偏小盘与高换手（实现成本敏感）、**任何截止日前的回测一律不可信**。这四条全部写入 §4 的策略规格与 §6 的评估协议。

### 2.3 尾部风险线：2024 年初 A 股微盘事故的教训

万得微盘股指数在 2024 年初的最大回撤接近 47%，同期沪深 300 仅跌约 2%；成因是小市值风格的拥挤交易、DMA 杠杆、雪球敲入与流动性挤兑的连锁共振，风格因子出现了十几倍标准差的极端波动，头部量化机构单月回撤 15–30%。业内复盘留下两句应写进任何系统的话：**统计规律不等于客观规律**；风险不在量价数据里，而在市场结构与拥挤度里。→ 对本项目的三条直接推论：①单市场+单风格集中是本系统必须用架构消灭的风险；②风险引擎必须包含**拥挤度与流动性**检查，不能只看波动率；③A 股个股不进入 v1 卫星层（无对冲工具、退出机制受限），观点可经 ETF 表达。

## 3. 设计裁决：对既往决策的逐项处理

| 裁决 | 内容 | 处理 |
|---|---|---|
| J1 | "选哪个市场"是**错误的问题**。正确的问题是：哪种结构能同时给出最多的不相关收益流+最全的降险工具。答案：**核心层=全球多资产（美元 ETF）**，一个境外券商账户即可全覆盖；**卫星层=美股个股**（LLM 信息优势在英文披露语料上最大、期权对冲与做空可得、无涨跌停/禁售约束） | v2.2 的美股方向**保留但重新奠基**（从"市场适配"升级为"多资产核心+美股卫星"）；美/港双市场适配简化——港股维持 icebox（11 篇 R2/R3 的复杂度依据不变） |
| J2 | **推翻 ADR-03 的绝对禁令**：LLM 可以作为 SignalProvider 产出**评分**（永远不是订单）。推翻依据=2.2 节两篇高质量证据+我们已建成 09 篇治理设施（当初禁令的核心理由是"无法验证"，如今验证机制已存在） | 新 ADR-22；保留不变：人工确认执行（ADR-09/12）、解释契约强制、仓位帽 |
| J3 | **LLM 回测污染升格为一等公民约束**：规则策略可用全历史回测；LLM/ML 信号的历史评估仅承认模型知识截止**之后**的样本，截止前结果一律标记 `contaminated` 仅供参考，最终去留由前向（影子/模拟盘）表现裁决 | 修订 06 §7、09 §2；我们已建的 L1/L2 阶梯恰好是该约束下唯一合法的评估器——架构幸运地提前对齐 |
| J4 | **风险引擎从"告警式"升级为"仓位控制式"**（这是"降低 risk"目标下最大的工程增量）：新模块 M8 | 见 §5；06/08 对应扩展 |
| J5 | 基础设施决策全部保留：账本/回放（07）、陷阱测试（06）、纪律引擎（03 §5.5）、上线阶梯（08）、模型治理（09）——它们与目标函数无关，是任何目标下的必需品 | 不变 |
| J6 | 策略库 v2 的 S1–S3（沪深 300 择时/红利低波/行业动量）整体退役，由 §4 的 v3 组合取代 | S3 的行业动量思想在卫星层保留（美股行业 ETF 版） |

## 4. v3 组合架构：核心–卫星–对冲

```
资金 100%
├─ 核心层 Core（默认 80%，委托人可在 70–90% 内定）
│   C1 全球多资产双动量（月频）：资产池 = 美股大盘 / 发达非美 / 新兴
│      / 美长债 / 美中期债 / 黄金 / 大宗 / 短债现金（各以最流动的美元ETF实现）
│      规则 = 相对动量（12-1 与 6-1 复合排序取前 3 等权）
│           × 绝对动量过滤（12月收益 < 短债则该仓位落入现金）
│   C2 趋势闸门：每资产 10 个月均线之下不持有（与 C1 绝对动量互为冗余保险）
│   C3 波动率目标：组合年化波动目标 10–12%，只向下调整股票类权重，不加杠杆
├─ 卫星层 Satellite（默认 20%，内部再分仓，每个 provider 有独立仓位帽）
│   S-A 美股行业 ETF 动量（12-1 前 3，月频）——规则型，可全历史回测
│   S-B LLM 财报方向评分（Booth 复制）：财报季对覆盖池逐家跑匿名化 CoT
│      评分，季频构建 ≥20 只分散多头（做空可行性在 L2 期评估），帽 10%
│   S-C LLM 新闻漂移评分（Lopez-Lira 复制）：每日全量新闻打分，取强信号
│      小组合，周频聚合以压换手；帽 5%，明示"衰减中的边"，随时可退役
└─ 对冲层 Hedge（0–5%，L3 后启动研究）：指数保护性认沽/领口策略的
    成本-保护曲线研究，作为熔断的期权化替代方案（研究票，非承诺）
```

**为什么这个结构最符合新目标函数**：核心层负责"降低 risk"（多资产分散消灭集中度、趋势闸门截断左尾、波动率目标控制尾部），其收益来源不依赖任何预测正确；卫星层负责"收益率"的上行尝试（LLM 证据线的两个可复制策略+一个规则策略），且每个都被治理阶梯、仓位帽和退役规则包裹——**卫星全灭时组合退化为一个稳健的全球配置，而不是归零**。这正是"别做完美但没用/一把梭哈然后爆炸"两个失败模式之间的最优解。

**账户前提（AS-6，需委托人确认）**：核心与卫星均以美元 ETF/美股实现，需要境外券商账户（IBKR 类）与合规出入金通道。若暂不可得，降级路径 = 以境内 QDII 跨境 ETF 近似核心层（覆盖窄、有溢价与额度风险，须如实建模），卫星层暂缓——此为落地的第一个检查点。

## 5. M8 风险引擎（新模块规格摘要，完整规格并入 03）

输入：持仓、行情、协方差估计（EWMA）、各资产 ADV。四个子机制，全部为**事前仓位控制**而非事后告警：①波动率目标器（周检，组合预测波动>目标 15% 即按比例降股票类权重，只减不加）；②趋势闸门（月检，C2 规则）；③**拥挤与流动性哨兵**（2024 教训的直接工程化：任一持仓的清仓量>其 ADV 的 20% 即标红并限制加仓；卫星层风格暴露对基准的偏离设上限）；④熔断阶梯（沿用 08 §5.5，新增：组合波动实现值连续两周>目标 1.5 倍 → 卫星层整体减半）。所有触发是账本事件，可回放。诚实条款：①的学术争议已在 2.1 节记录，本系统只用其减仓方向，预期收益贡献登记为"未知，可能为负（踏空成本）"，其存在理由是回撤控制。

## 6. 评估协议修订（并入 06/09）

规则策略（C1/C2/C3/S-A）：全历史回测 + walk-forward，照旧。LLM 策略（S-B/S-C）：**离线评估只承认所用模型知识截止后的窗口**（若用 2025 年模型则历史窗口极短，如实接受）；主要证据来自 L1 影子 ≥40 交易日 + L2 模拟盘 ≥60 交易日的前向记录；准入线沿用 09 §2，另加两条退役触发器——滚动衰减监测（信号 IC 的时间趋势显著为负且 CI 上界触零）与拥挤度外因（该类策略公开采用度激增的定性评审）。**明示：S-B/S-C 的"历史高夏普"引自论文，不构成我们的预期；我们的预期在前向数据出来之前为"未知"。**

## 8. 新增 ADR（草案，拍板后入 02 篇）

ADR-22 LLM 可产出评分型信号，禁产订单，须过治理阶梯（推翻 ADR-03 的绝对禁令部分，保留其"不直接下单"内核）。ADR-23 组合结构确立为核心–卫星–对冲，核心=全球多资产美元 ETF。ADR-24 LLM/ML 信号评估仅承认知识截止后样本，截止前回测标记 contaminated。ADR-25 风险管理升级为事前仓位控制（M8），波动率目标仅用于减仓方向。ADR-26 拥挤度与流动性哨兵为强制风控维度（2024 微盘事故教训）。ADR-27 税务建模冻结（委托人指令，风险自担备案），钩子保留。ADR-28 A 股个股不入 v1 卫星层，观点经 ETF 表达；港股维持 icebox。

## 9. 需要委托人拍板的清单（按重要度）

1. **AS-6 账户前提**：是否已有/可开境外券商账户？（决定核心层走正版还是 QDII 降级版）
