## 1. 一页纸摘要

**yquant 是什么**：以**全球宏观信息为雷达、美股与美国上市 ETF 为唯一执行工具**的个人投资决策系统。使用者在 Selfwealth 手动下单；系统负责回答三个问题——**机会在哪、风险在哪、何时买卖是明智的**——并以事件账本保证每个答案可逐字节还原。
**三层组合**：核心层 75%（全球多资产 ETF：双动量+趋势闸门+波动率目标，职责=降风险）；系统化卫星 15%（行业动量+LLM 财报评分+LLM 新闻评分，职责=收益尝试）；战术机会层 10% 硬上限（宏观机会簿+2x 条款，职责=把全球判断变成有失效条件的美股表达）。
**LLM 的位置**：十个岗位（§8），全部隔着 schema 校验与规则预算两层护栏；可产评分、永不产订单（ADR-22）；知识截止前的回测一律标记 contaminated（ADR-24）。
**不做什么**：反向 ETF 与 3x（首年，icebox）；期权（研究票除外）；做空；日内；自动下单；税务建模；A 股/港股直接交易（中国观点经 MCHI/FXI/KWEB 等美上市 ETF 表达）；多币种（全系统 USD）。

## 2. 目标、指标与范围

### 2.1 目标函数
最大化长期净值口径风险调整收益：**主指标卡玛比率**（年化收益/最大回撤），辅指标夏普、月度亏损尾部。交易成本与滑点照常建模；LLM 用量记账但无预算上限；税务不建模（委托人明示排除，风险自担备案）。

### 2.2 用户画像
个人投资者一名，Selfwealth 账户手动执行美股/ETF 交易（现金账户，USD），周/月频调仓为主，日常盯盘 <30 分钟，持有 OpenAI 兼容 LLM API。

### 2.3 用户故事（验收终点）
US-1 每个美股交易日收盘后，我打开"今日简报"：全球天气面板（状态机四态+五支柱）、宏观事件卡、自选股事件卡、今日必看 Top3。5 分钟读完。
US-2 每周日"机会-风险委员会"产出：Top5 风险仪表盘（含我的暴露度+可交易防御表达）、机会簿（每条带入场与失效条件）、核心层倾斜建议。
US-3 任一建议单需过 6 项 checklist（含红队反方陈述）才能标记已执行；我在 Selfwealth 下单后回填成交，系统闭环滑点。
US-4 每笔战术持仓的命题由 Thesis 哨兵每日体检；"已失效"自动生成平仓建议。
US-5 我说"重演 6 月 3 日的决策"，`replay --strict` 逐字节还原当日全链路。
US-6 回测实验室里任何策略报告强制含：与 SPY 买入持有对照、成本三档、样本外 walk-forward、warnings。

### 2.4 成功指标（90 天检验）
周活跃 ≥4 天；每日投研 ≤10 分钟且 severity≥4 事件零遗漏；交易留痕率 100%、计划外交易占比下降；回放一致率 100%；机会簿失效条件触发的止损执行率 100%；状态机与演习判据见 06/08。

### 2.5 非目标（写入合同）
见 §1"不做什么"。新想法一律进 `docs/icebox.md`（现有 icebox 项：反向 ETF、3x、期权对冲腿、半自动执行、A 股卫星、多币种）。

## 3. 组合架构（含硬性预算规则）

```
资金 100%（USD）
├─ 核心层 Core 75%
│   C1 全球多资产双动量（月频）：池={VTI/SPY, VEA, VWO, IEF, TLT, GLD, PDBC, BIL}
│      相对动量（12-1 与 6-1 复合排序取前 3 等权）×绝对动量（12月<BIL→该腿落现金）
│   C2 趋势闸门：任一资产收盘<10月均线→该资产不持有（与C1绝对动量互为冗余）
│   C3 波动率目标：组合年化波动目标 10–12%，仅下调风险资产权重，不加杠杆
├─ 系统化卫星 Satellite 15%
│   S-A 行业ETF动量（SPDR 11行业+SMH+XBI，12-1 前3，月频）
│   S-B LLM财报方向评分（Booth式匿名化CoT，季频，≥20只分散多头，帽10%）
│   S-C LLM新闻漂移评分（Lopez-Lira式，日打分周聚合，帽5%，标注"衰减中的边"）
└─ 战术机会层 Overlay ≤10%（硬上限，ADR-35：配置只可下调；上调=变更+重签区间书）
    宏观机会簿持仓（每仓帽5%）＋ 2x多头条款仓位（见§7）
    防御表达（无反向工具）：减仓 → BIL/SGOV现金、IEF/TLT久期、GLD黄金
```

**预算规则（代码强制）**：三层预算独立核算，跨层挪用被拒；Overlay 合计市值/组合净值 >10% 时新建仓 proposal 自动拒绝并记 risk_event；同向机会合并计算暴露。

## 4. 系统架构与模块总览

M1 数据基座 → M2 回测引擎 → M3 策略库；M1 → M4 个股事件卡 + M9 宏观雷达；M3/M4/M9 → M5 纪律引擎 → 人（Selfwealth）；M8 风险引擎横切消费 M9 状态并控制 M2/M5 仓位；M6 UI；M7 调度；账本与回放（07 篇）为全体底座。单向依赖，禁止跨层直连。

## 5. 模块规格

### 5.1 M1 数据基座
**交易域数据**：美股/ETF 日线（yfinance 主、Stooq 备），`auto_adjust` 语义写死为"调整价与原始价双存"；EDGAR 公告（8-K/10-K/10-Q/Form4/13D-G，合规 User-Agent，AS-1'）；指数 ^GSPC/^NDX。**信息域数据（全球）**：FRED（DGS2、DGS10、T10Y3M、BAMLH0A0HYM2 高收益 OAS、NFCI、DTWEXBGS 美元）、全球股指（^N225/^GDAXI/^FTSE/^HSI/^KS11）、汇率（USDJPY=X、EURUSD=X、USDCNH=X、DX-Y.NYB）、商品前月（GC=F/CL=F/HG=F）、波动（^VIX/^VIX3M/SKEW）、GPR 与 EPU 指数（官方免费下载）、CFTC COT、AAII（证据级 C 旁证）。全部走适配器+落盘+manifest（ADR-06/13）。
**Universe 与幸存者偏差（ADR-18）**：ETF 策略天然免疫；个股 watchlist ≤30 只用于事件卡与 S-B/S-C；若未来做个股宇宙策略，必须用 S&P 500 历史成分变更表构造时点宇宙（AS-2'：成分变更数据源验证），行情缺失个体在报告中计数披露。
**质检**：K 线覆盖率对照 XNYS 日历、异常跳变（|Δ|>20% 且非拆股日）、调整价连续性、公告时间戳非未来、FRED 序列新鲜度。验收：质检通过率 ≥99%，断源降级链路 mock 测试通过。

### 5.2 M2 回测引擎（backtrader + 美股约束层）
**约束**：无当日禁售；**交收=资金可用性**：现金账户 T+1 交收，未交收资金再买入触发 GFV 计数（T6 测试）；停牌/halt 日零成交（T3'）；无涨跌停撮合；最小单位 1 股（默认整股）。**成本模型（config 默认）**：`per_trade_fee_usd`（Selfwealth 固定费，AS-8 以现行费率为准，默认 9.5）、SEC fee 与 TAF（卖出侧，按现行微量费率参数化）、滑点 0.05%（ETF）/0.10%（个股）。**报告强制项**：SPY 买入持有对照、成本 0/1x/2x 三档、walk-forward 样本外、参数敏感性、warnings、拒单明细。验收：T0–T18 相关项全绿、确定性、全池十年月频 ≤10 分钟。

### 5.3 M3 策略库
C1/C2/C3 与 S-A 按 §3 规格实现，每策略交付 spec 文档+walk-forward 报告+诚实结论（负面结论不扣分）。S-B/S-C 见 §5.9 与 09 篇治理阶梯（评估协议 ADR-24：只认知识截止后样本+前向）。

### 5.4 M4 个股事件卡（AI 简报）
分类法采用官方语汇（ADR-21）：8-K item 编码直读、Form 4 结构化解析**不走 LLM**、13D/G、财报（10-Q/10-K 读 MD&A 与 Risk Factors 节选，长文按需深读——无预算上限但仍记账）。幻觉防护不变：数字回验、原文链接必填、schema 强校验、宁可"不确定"不可编造。评测集：英文 filings 120 条重建（覆盖 8-K 主要 item），验收线同 06 篇。

### 5.5 M5 纪律引擎
Proposal 增加必填字段：`invalidation_condition`（可机读失效条件，缺失即拒收，ADR-30）与 `red_team_note`。**Checklist 六项**：①既定规则触发（否则填计划外理由）②非冷静期③单仓/层预算合规④回撤状态允许⑤已阅当日红旗⑥**已阅红队反方陈述**。执行闭环：确认→Selfwealth 手动下单→回填成交价/股数/费用→系统计算滑点入 metrics。风控规则：单票 ≤15%（Overlay 单仓 ≤5%）、行业 ≤35%、回撤 10% 禁加仓/15% 触发阶梯（§5.8）、连亏 3 笔冷静期 3 日。AI 复盘周报按"状态/规则/运气"三分归因。

### 5.6 M6 UI（Streamlit 六页）
今日简报（全球天气面板+事件卡流+Top3）｜机会与风险（风险仪表盘+机会簿+Thesis 哨兵状态）｜组合与风控（三层预算水位、净值 vs SPY、回撤、risk_events）｜回测实验室｜交易台账（checklist 交互+滑点闭环+周复盘）｜设置与系统健康（P 指标、数据新鲜度、任务日志、LLM 用量）。

### 5.7 M7 调度
`exchange_calendars`（XNYS）锚定：收盘后 30 分钟数据更新→简报流水线→推送；周日委员会；业务时区 America/New_York，存储 UTC ISO8601，夏令时切换由日历库处理（T10 测试）。通知渠道可配置（飞书默认，可换 Telegram/邮件）。

### 5.8 M8 风险引擎（事前仓位控制，ADR-25/26）
①波动率目标器（周检，EWMA 协方差，超目标 15% 按比例降风险资产，只减不加）；②趋势闸门执行器（月检 C2）；③拥挤与流动性哨兵（持仓清仓量>20 日 ADV 的 20% 标红限加仓；卫星层风格暴露偏离上限）；④熔断阶梯：回撤 ≥10% 禁加仓 → ≥15% Overlay 清至防御、核心层收至波动目标下限 → 状态机 Crisis 同步触发 §5.9 否决。全部触发为账本事件。

### 5.9 M9 全球宏观雷达（三层判断）
**Layer1 状态机**（规则化、全历史可回测）：五支柱（趋势/信用/波动结构/广度/宏观流动性，参数与证据级见 06 附表）加权合成，滞回 2 周，输出 RiskOn/Neutral/RiskOff/Crisis；对 LLM 层持否决权（ADR-32）：Crisis 态 Overlay 仅允许防御方向，2x 强制清零。
**Layer2 宏观事件流**（LLM）：MacroEventCard{source_type, hawk_dove∈[-2,2]（央行文本，五档对齐 Hansen–Kazinnik 体系，季度偏鸽校准）, channels 传导渠道, us_expression_map, magnitude 1–5, half_life, confidence, evidence_urls, contrarian_note}；severity≥4 推送并可触发临时委员会。
**Layer3 机会-风险委员会**（周日+触发式，分析员→红队→合成三步）：输出风险仪表盘 Top5（含组合暴露度+防御表达）、机会簿（命题≤100字/全球依据/美股 ticker/入场条件/**失效条件必填**/权重/时限/红队陈述）、核心层倾斜建议（±10% 相对带宽内）。确定性护栏：层预算、状态机否决、失效条件强制、同向合并。

## 6. 买卖时机准则目录（Timing Doctrine）
**买入**：R1 再平衡日机械执行（A）；R2 恐慌再入——`VIX>30 且 VIX/VIX3M 倒挂 且 HY OAS 三月变化<+150bp 且 200日线上比例<20%` → 战术现金分三批隔周回补宽基（B；机制=区分波动事件与信用事件，证据"有效但不一致"如实标注）；R3 趋势恢复确认（收复 10 月线且离开 RiskOff）恢复核心配置（A−）；R4 机会簿入场（条件机读为真+红队未否决；前向验证中，C+）。
**卖出**：S1 失效即卖（Thesis 哨兵日检，命题死亡为第一卖出依据）；S2 状态降级阶梯（RiskOff→Overlay 多头减半+2x 清零；Crisis→仅防御）；S3 时间止损（战术仓 >60 天无推进证据强制复审默认平）；S4 再平衡回带（核心资产偏离目标 >25% 相对值）。
**永不做**：无失效条件建仓；亏损战术仓无规则加码；Crisis 加杠杆；用 2x 表达长期观点。

## 7. 杠杆条款 v3.1（ADR-31，委托人已确认）
仅允许高流动性 **2x 多头** ETF（示例 SSO/QLD/UGL；准入门槛同 §5.1 且 20 日均成交 >$5000 万）；**合计 ≤组合 5%，单仓 ≤3%**，计入 Overlay 预算；开仓三条件：状态机=RiskOn ∧ 标的>10 月线 ∧ VIX<20；状态降级即强制清零；每仓 60 天强制复审；建仓即写失效条件；风险预算按名义敞口 ×2 折算。**机制诚实条款**（写入 UI 提示）：日重置复利，长期持有≠2 倍指数，高波动震荡市有显著波动损耗。3x 与反向 ETF：icebox，一年后凭 L2/L3 账本数据重议。

## 12. FAQ（增补项）
**Q 为什么首年没有反向 ETF 对冲？** 委托人决策（ADR-31）。防御哲学=减法：降敞口、加现金/久期/黄金，历史上足以覆盖个人组合的风险管理需求，且避免反向工具的路径损耗与误用。**Q 为什么不用期权？** 研究票（icebox），L3 后凭数据立项。**Q Selfwealth 手动下单如何不破坏"高精度"？** 执行回填是账本事件，滑点闭环量化"人的执行误差"，模拟盘孪生账户分离假设误差与执行误差（08 篇）。**Q 状态机会不会频繁打脸？** 会有踏空成本（2020 型 V 反弹已写入演习）；它的职责是回撤控制，收益贡献登记为"未知，可能为负"，这是买保险的定价。**Q 中国/日本观点怎么表达？** 美上市国别 ETF（MCHI/FXI/KWEB/EWJ），或转化为风险仪表盘的暴露调整。
