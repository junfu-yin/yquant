## 30 秒版结论

v1 方案的"数据→预测→信号→AI 自主交易"主线经论证不成立（对手盘、时效、回测欺骗、尾部风险四重攻击，见 02 实验 1）。v2 重新定位为**"AI 投研副驾 + 纪律引擎"**：LLM 负责每天读完你读不完的公告新闻并可溯源地结构化（不给买卖建议），回测系统负责在花真钱前杀死坏想法（内置 A 股约束与陷阱测试），纪律引擎负责把规则变成强制流程对抗行为偏差。总工作量约 35–46 人日，六个里程碑各有硬性出口判据，并预设了三条项目级止损判据（K1–K3）防止"做完美但没用"。
