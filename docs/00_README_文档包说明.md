# yquant 项目文档包 · 总览（v3.1）

> 生成日期：2026-07-05 · 状态：v3.1 定稿（全球宏观视角 × 美股执行 正式版）
> 本包为**当前唯一有效文档集**，取代仓库中全部旧版（归档方式见《05 废止与归档清单》）。

## 一句话定位

yquant 是一套**以全球宏观信息为雷达、以美股与美国上市 ETF 为唯一执行工具、由使用者在 Selfwealth 手动下单**的个人投资决策系统。它回答三个问题：机会在哪、风险在哪、何时买卖是明智的；LLM 承担信息压缩、宏观研判、alpha 尝试与决策卫生共十个岗位，但每个输出都隔着 schema 校验与规则预算两层护栏，扣扳机的永远是人。

## 组合结构（委托人已确认）

核心层 75%（全球多资产 ETF 双动量+趋势闸门+波动率目标）／系统化卫星 15%（行业动量+LLM 财报评分+LLM 新闻评分）／战术机会层 **10%（硬上限，只可下调）**。首年杠杆权限：仅高流动性 **2x 多头** ETF（合计 ≤ 组合 5%），无 3x、无反向；防御一律用减法（减仓→现金/久期/黄金）。

## 版本沿革（30 秒版）

v1（原始方案）：数据→预测→AI 自主交易 → 经论证不成立。v2：A 股投研副驾+纪律引擎。v2.2：美/港适配（部分正确，理由错误）。v3：目标函数重设为收益率×降风险，核心-卫星架构，LLM 评分解禁（治理阶梯内）。**v3.1（本版）**：交易域收敛为美股+美上市 ETF（Selfwealth 手动执行），信息域扩展为全球宏观，新增 M9 宏观雷达（状态机/事件流/机会-风险委员会）、战术机会层硬上限条款、首年杠杆条款、四场景历史演习强制集；删除 A 股优先、港股适配、税务建模、LLM 绝对禁令等已冲突内容。
