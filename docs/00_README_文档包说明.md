## 30 秒版结论

v1 方案的"数据→预测→信号→AI 自主交易"主线经论证不成立（对手盘、时效、回测欺骗、尾部风险四重攻击，见 02 实验 1）。v2 重新定位为**"AI 投研副驾 + 纪律引擎"**，研究标的为**美股（主）与港股（辅）**：LLM 负责每天读完你读不完的公告新闻（SEC EDGAR / 披露易 HKEXnews）并可溯源地结构化（不给买卖建议），回测系统负责在花真钱前杀死坏想法（内置美股/港股真实交易约束——交收周期、LULD/熔断、VCM/CAS、费用——与陷阱测试），纪律引擎负责把规则变成强制流程对抗行为偏差。

**v3 增量（依据 12 篇 SSOT）**：把模糊的"有用"收敛为**单一目标函数——风险调整收益（主指标卡玛=年化收益/最大回撤，辅以夏普与月度亏损尾部）**；组合结构确立为**核心–卫星–对冲**（核心=全球多资产美元 ETF 降险，默认 80%；卫星=美股个股博上行，默认 20%；对冲=研究票 0–5%），卫星全灭则退化为稳健全球配置而非归零；风险管理从"事后告警"升级为**事前仓位控制 M8**（波动率目标器 + 趋势闸门 + 拥挤/流动性哨兵 + 熔断阶梯，只减不加、无杠杆）；**LLM/ML 从"绝对禁产信号"松绑为"产评分不产订单"**（推翻 ADR-03 禁令部分、保留不下单内核，评估仅承认知识截止后样本 J3/ADR-24）；按委托人指令**冻结税务建模与 LLM 日预算硬上限**（`tax_*` 钩子保留、用量记账保留，交易成本与滑点照常建模）。总工作量约 78–102 人日（阶段一 35–46 + 阶段二 43–56），里程碑各有硬性出口判据，并预设三条项目级止损判据（K1–K3）防止"做完美但没用"。
