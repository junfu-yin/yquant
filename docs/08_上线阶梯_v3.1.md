# 08 · 模拟盘与小资金上线方案（v3.1）

> 取代旧 08。阶梯哲学与"机器硬门槛/收益预注册"双轨制**不变**；本版适配美股规则、USD、三层组合与委托人三项确认。◆ 为变更点，未标记条款沿用旧 08 原文。

## 1. 五级阶梯（◆ 金额与对象更新）
L0 回测 → L1 影子（≥20 交易日；◆ 含"纸上机会簿"运行）→ L2 模拟盘（◆ 虚拟 **$50,000**，系统全自动，≥60 交易日且 ≥2 次月度调仓且 ◆ **四场景历史演习通过**）→ L3 小资金试点（真实小额，◆ Selfwealth 手动执行，≥6 个月；收益评估 ≥12 个月）→ L4 半自动执行（icebox）。

## 2. PaperBroker（◆ 美股规格）
与回测引擎共享同一约束实现：T 日收盘后 proposal→T+1 开盘撮合，成交价=开盘×(1±滑点) 且 ∈[low,high]；halt 日拒单；◆ 交收账簿：T+1 交收、未交收资金标记、GFV 计数（T6）；◆ 固定费+SEC/TAF 费用模型；股息按除息/付息日入账（USD）；日终 P1/P2 对账分毫必平，不平即 S1 并冻结次日自动下单。诚实性由 T7 双引擎 parity 结构性保证。◆ 三层预算与 2x 条款在模拟盘全量生效（T17/T18 在真实数据流上复验）。

## 3. 数字孪生（不变）
L3 起模拟账户继续并行，用于分离"假设误差"与"执行误差"；◆ 执行误差=Selfwealth 手动成交回填 vs 孪生假设，逐笔滑点入 metrics，月度《执行质量报告》。

## 4. 预注册预期区间书（◆ 分层版）
升 L3 前签署、入账本、改动即新版留痕：
- **核心层**：给出 walk-forward 样本外的年化/波动/最大回撤 10–90 分位区间（可给，规则策略）+踏空成本预期（趋势/波动目标的保险费）。
- **卫星层 S-B/S-C**：◆ 明示"论文证据不构成我们的预期；本层预期=未知，由前向数据形成"；仅预注册**过程指标**（覆盖率、弃权率、评测线维持）。
- **Overlay**：◆ 预注册过程指标——机会命中率基线（纸上机会簿期实测值）、失效止损执行率=100%、红队否决率区间、状态机否决遵从率=100%。
- 通用条款：实盘滚动结果出区间（**含好得反常**）→ 触发调查；"通过 L2 ≠ 策略有效"声明不变。

## 5. L2→L3 升级 Gate（◆ 增补）
机器硬门槛（一票否决）：P1–P11 近 30 日全绿；部署 tag 上 T0–T18 全绿；◆ **四场景历史演习报告齐备且评审通过（ADR-36）**；P0/P1 清零且各沉淀新测试;红旗零遗漏；一次消防演习通过；抽 3 run 严格回放一致；◆ 纸上机会簿 ≥8 周统计（命中率/止损有效性/否决率）交付。决策门槛：K1' 复核+委托人 human_action 入账。
试点资金与仓位纪律：金额委托人书面定（"全亏无碍"额度）；首月 Overlay 减半运行（≤5%）、2x 权限关闭，次月起按条款恢复。

## 6. L3 熔断与回退（◆ 防御表达更新）
组合回撤 ≥10% 禁加仓；≥15% **降回 L2**（真实账户只减不增）；◆ 防御动作序列（无反向工具）：①Overlay 多头清至现金 ②2x 清零 ③核心层收至波动目标下限 ④超防御需求以 BIL/IEF/GLD 表达；单月 ≥2 起 P0 暂停新仓；区间书越界（双向）触发调查；滑点+费用 >2 倍假设 → 停新仓检修执行约定（如改用限价单/更换下单时段）；K3' 使用度冻结条款沿用。

## 7. 运行手册目录（◆ 增补）
